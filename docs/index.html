Welcome

This tutorial introduces you to some of the work we do at SomerStat. The data we will be using comes from our annual Happiness Survey (insert description of this here), which this website reads from the Open Data Portal. We will introduce you to some of what we use R for. According to the notably reliable Google Search AI, “R is useful for cleaning, importing, and visualizing data.” We mostly use R for the first two of these use-cases.

There is a good chance that your time here will include analyzing data in R and/or reading/writing data to/from the Open Data Portal, so I hope this little tutorial is more helpful than harmful.
